# Manual de Usuario: Cotizador Solar

## Introducción
El cotizador solar permite generar propuestas técnicas y comerciales para instalaciones solares basadas en el consumo eléctrico del cliente.

## Beneficios
- Cálculos automáticos de sistema solar
- Generación de propuestas PDF
- Dos modos de operación (rápido y avanzado)
- Personalización de parámetros técnicos

## Paso a Paso

### 1. Crear nueva cotización
- Menú: Cotizador Solar > Cotizaciones > Nuevo
- Completar datos del cliente (Cliente, Ubicación, Medidor, NAC)

### 2. Ingresar consumo eléctrico
- Proporcionar valores mensuales de consumo (12 meses) en pestaña "Historial de Consumo"

### 3. Seleccionar modo de cálculo
#### Modo Rápido
- Ideal para estimaciones preliminares
- Campos requeridos:
  - Horas Sol Pico (HSP)
  - Precio por watt
  - Watts por panel
  - Mínimo de paneles

#### Modo Avanzado
- Para cálculos precisos
- Campos adicionales:
  - Eficiencia del sistema
  - Número de paneles
  - Modo de precios
- Usar botón "Auto-Ajustar" para cálculo automático

### 4. Generar propuesta
- Hacer clic en "Generar Propuesta"
- El sistema creará un PDF con:
  - Datos del cliente
  - Consumo eléctrico
  - Especificaciones del sistema solar
  - Detalles de ahorro

## Consejos de Uso
- Verifique los valores de consumo antes de calcular
- Use "Auto-Ajustar" en modo avanzado para optimizar el número de paneles
- Revise siempre la propuesta PDF antes de enviar al cliente
